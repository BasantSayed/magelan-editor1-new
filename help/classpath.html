Setting the ClassPath
Menu:   Preferences | ClassPath

Basics

A list of paths to compiled code (class files and JAR files) is called ClassPath. The application will sometimes need the user to explicitly declare the location of a JAR file or class files. This is done using the ClassPath manager dialog.

Adding ClassPath entries

You could either type the path entry you like to add or select the file or directory using the Add button.

If an entry cannot be found an error icon will appear next to it. If an entry can be found but is not a path or JAR file again an error icon will appear.
