Using Help

Help contents

The editor has two separate help systems - one is the built-in help system, which is intended for end-users; the other can be started by opening the ./doc/README.html file and is intended for developers.

The built-in help system describes how to use the editor and its extensions. The help system for developers addresses technical issues and is provided to assist developers in creating custom drawing entities or editor extensions.

Help on extensions

When an extension is loaded the editor will automaticly try to locate its help file. The help files of all loaded extensions are listed in the Extensions tab of the help window. If the editor is unable to locate the help file for an extension an error icon appears next to extension's name.

The location of the help file of an extension is declared in extension's manifest file. For more information on manifest files and extensions please refer to the Using Extensions section.

Modifying help contents

To modify a help topic, press the right mouse button on it. From the popup menu select Edit to modify the help item. To add user notes or new help items select the relevant options from the popup menu.

Note: There are presently some known issues with modifying the help contents. To see them please refer to the Known Problems section.
