Known Problems

The present 1.0 RC4 version of the software contains a number of known problems. Some of the more important issues are listed below.

Drafting

During the process of creating some entities they are not shown on the screen until the user completes their creation and clicks with the mouse on the drawing.

When loaded some drawings will suppress snapping.

Under certain circumstances zoom extensions may function incorrectly.

Serialization

Binary serialized drawings (the default file format) are not compatible with the files saved by the previous release. As of the next release all issues with binary serialized drawings will be solved.

Help System

When a hyperlink from the help screens is clicked on the selected item from the tree view does not change.

When a note or help item are added to the help tree (or deleted) the view is not refreshed and the change is not immediately visible. To see such changes, close the help system and then open it again.

Language

French translation is incomplete for this release. Use of English is recommended.
