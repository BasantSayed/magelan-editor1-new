Overview

The Magelan project aims to provide a set of easy to use and incorporate Java 2D vector graphics editor components.

It is divided into two parts:

  • core

    The core library defines powerful drawing and editor models, as well as basic viewer and editor Swing components. It is intended as a platform for the development of graphics editors and related products.

    The library is also very convenient to use for all kinds of visualisation tasks and in applications where simple graphical user input is required. If more complex behavior is required, the editor library should be used.

    The editor Swing component features selection and drag and drop editing of entities (stretching and moving).

    The core comes with a set of pre-defined entities. It includes not only basic entities like lines, circles, polylines, etc. but a number of more sophisticated ones like dynamic labels, which display a chosen property of an entity; block definitions and views.

    Finally, the offered drawing and editor models are not only powerful and flexible, but very easy to use and extend, and new entities are simple to write.

  • editor

    The editor uses and further extends the drawing and editor models, defined in the core library, and provides a basic stand-alone editor implementation.

    A number of editor extensions (tools) are included in it. They allow the user to create and edit graphical entities, as well as to navigate through the drawing and perform various operations on it.

    The editor supports a number of file formats, most notably it is able to read and write basic DXF files. It can also export drawings to EPS and SVG formats.

    Extensions are very easy to create, maintain and plug into the editor component.

Note: For more information regarding the editor, please refer to the documentation bundled with it.

Distribution Package

The structure of the distribution package is as follows:

  • bin folder, containing magelan-core.jar with the compiled and ready for use library;
  • src folder, containing all source files;
  • doc folder, containing all documentation, including JavaDocs;
  • lib folder, containing all necessary libraries;

To see a demo of the library, execute the magelan-core.jar file.

Note: the magelan-commons.jar file from /lib must be placed in the same folder as the magelan-core.jar file.

JDrawing
The project features a simple to use but very flexible and easily extendible vector graphics framework. It allows efficient rendering of thousands of entities. It is inherently similar to the core library. To bring the similarities further, some of the excellent design solutions of JDrawing were implemented in this library.
