General Information

The primary goals of the Magelan project are to establish a compact and flexible drawing editor API and a modular and powerful editor environment. A lot of efforts have been put into keeping the drawing model and extensions framework as simple and convenient to use as possible. Custom drawing entities and extensions for the editor application are easy to write and plug in the editor environment.

The project is divided into two parts - one is the core library defining clear and powerful drawing and editor models, and the second is the editor library, which extends the core and features a basic scalable editor implementation. This is a complete release and includes both libraries.

Notable Features

  • full control over all entity properties;
  • easy plugability of entities, editor extensions, templates etc.
  • reading and writing basic AutoCAD DXF files;
  • export to SVG and EPS formats;
  • multi-language support;
  • comprehensive help system.

Requirements

The project distribution package requires either the Java Runtime Environment or the Java SDK. They can be downloaded at http://java.sun.com/j2se/downloads.html. Also note that the application needs Java 1.5.0 or higher and will not run (correctly) under older versions.

Magelan has been developed and tested mostly on Windows.

Distribution Package

There are three distribution packages: for Windows with JRE, for Windows without JRE and for all other systems.

The structure of the distribution packages is as follows:
/src containing all editor source files;
/doc containing the available documentation, including JavaDocs;
/extension containing extensions and extension manifests;
/lib containing the compiled core and other libraries;
magelan.jar an executable JAR file, placed in the root folder of the package and containing the compiled editor.

To run the editor, execute the magelan.jar file.
